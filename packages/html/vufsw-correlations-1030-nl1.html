    Question

    Stel je voor dat we een sterke positieve correlatie zouden vinden tussen het aantal politieagenten en de misdaadcijfers in hetzelfde gebied. We zouden kunnen veronderstellen dat deze relatie wordt verklaard door de bevolkingsomvang van dit gebied. Welk type onderzoeksdesign beschrijft dit het beste?


    1. TRUE: Spurious correlatie met misdaadcijfer als afhankelijke variabele, bevolkingsomvang als spurious variabele en aantal politieagenten als onafhankelijke variabele
    2. FALSE: Mediatie met misdaadcijfer als afhankelijke variabele, bevolkingsomvang als spurious variabele en aantal politieagenten als onafhankelijke variabele
    3. FALSE: Mediatie met aantal politieagenten als afhankelijke variabele, bevolkingsomvang als spurious variabele en misdaadcijfer als onafhankelijke variabele
    4. FALSE: Spurious correlatie met aantal politieagenten als afhankelijke variabele, bevolkingsomvang als spurious variabele en misdaadcijfer als onafhankelijke variabele

    Solution

    This is an example of a spurious correlation where the crime rate as dependent variable, population size as spurious variable and the number of policemen as independent variable. The path diagram below illustrates this.


    1. True
    2. False
    3. False
    4. False
