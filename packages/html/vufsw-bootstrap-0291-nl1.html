    Question

    Kijk naar het onderstaande mediatiemodel. Waarom is de bootstrap methode de beste methode om het mediatie-effect te toetsen?


    1. TRUE: Omdat de bootstrap methode gebaseerd is op de toetsing van het indirecte effect αb en niet het verschil tussen het totale effect c en het het directe effect c’ en omdat de bootstrap methode betrouwbaar is ook bij kleine steekproeven
    2. FALSE: Omdat met de bootstrap methode is het indirecte effect altijd gelijk aan zowel αb als c - c’ en omdat de bootstrap methode betrouwbaar is ook bij kleine steekproeven
    3. FALSE: Omdat de bootstrap methode gebaseerd is op de toetsing van het indirecte effect αb en niet het verschil tussen het totale effect c en het het directe effect c’ en omdat met de bootstrap methode is het indirecte effect altijd gelijk aan zowel αb als c - c’
    4. FALSE: Omdat de bootstrap methode gebaseerd is op de toetsing van het indirecte effect αb en niet het verschil tussen het totale effect c en het het directe effect c’ en omdat de bootstrap methode betrouwbaar is ook bij grote steekproeven

    Solution

    De bootstrap methode en de Sobel toets zijn beter dan de Baron en Kenny methode omdat ze een meditatie-effect toetsen
    door een directe toets uit te voeren voor het indirecte effect αb. De Baron en Kenny is een nogal achterhaalde methode die
    een mediatie-effect toetst door het verschil tussen het totale en het directe effect c-c’ en waarbij aan 4 condities voldaan moet worden. De eerste conditie (i.e. het bestaan van
    een totale effect) is eigenlijk niet nodig. Daarnaast legt de Baron en Kenny methode te veel nadruk op statistische significantie.
    Daarnaast is de bootstrap beter dan de Sobel toets omdat de Sobel toets een puntschatting voor αb gebruikt. Daarvoor moet de steekproef relatief
    groot zijn zodat aan de assumptie van multivariate normality is voldaan voor αb.


    1. True
    2. False
    3. False
    4. False
