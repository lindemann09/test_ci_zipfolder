    Question

    Stel, we hebben een onderzoek uitgevoerd en een regressiemodel geschat om na te gaan welke factoren de mentale gezondheid onder werknemers beïnvloeden. Er wordt voorspeld dat hoge psychologische eisen, gebrek aan controle over het werk, en gebrek aan sociale ondersteuning op het werk allemaal de mentale gezondheid verminderen.

    We meten “mentale gezondheid” (Y) op een schaal van 0-100, waarbij een hogere score een betere mentale gezondheid betekent. Voor de kenmerken van de baan die iemand heeft, maken we een onderscheid tussen “psychologische eisen” (X1), “gebrek aan controle” (X2) en “gebrek aan sociale ondersteuning” (X3). We meten elke onafhankelijke variabele op een schaal van 0 (laag op dit kenmerk) t/m 4 (hoog op dit kenmerk).

    We vinden de volgende regressievergelijking (zie onder) :
    Wat is de alternatieve hypothese voor het effect van psychologische eisen op iemands mentale gezondheid?


    1. FALSE: Ha: β = 0
    2. FALSE: Ha: β > 0
    3. TRUE: Ha: β < 0
    4. FALSE: Ha: β ≠ 0

    1. False
    2. False
    3. True
    4. False
